# ようこそあんこ鯖へ!


## What is Anko Server??

あんこ鯖とは、MinecraftPE/BEで経済を本格的に楽しめるサーバー・TOKYOSERVERから派生した最新経済サーバーである。

尚、TOKYOSERVERとは開発者や運営は同じだが、worldなどは一切引き継がれておらず新しい生活を楽しむことができる。


## あんこ鯖の特徴

あんこ鯖では生活・整地・カジノ・PVP・PVEという4大要素を取り入れた楽しいSERVERである。

各worldに分かれて様々なミニゲームを楽しみ、時にはカジノで破産することも可能。

それぞれの村でほのぼの農業生活。
