# ようこそ

![](https://i.azisaba.net/top.webp)

アジ鯖(Azisaba Network) とは、鯖主の [ledlaggazi](https://twitter.com/azi_blueline) 氏と [多数の運営陣](/staff) により運営されるJava版マインクラフト(通称マイクラ)のサーバーである。

ちなみに2013年頃に存在した 鯵鯖 というサーバーとは全くの無関係である。

2018年に本格的に活動を開始し、現在ではたくさんのプレイヤーに遊んでいただける大規模なサーバーへと成長することができた。

これからもアジ鯖は成長を続けていきますので、応援よろしくお願いします。
